# 🏠 Real Estate Booking Platform

A modern full-stack **Real Estate Property Booking Platform** where users can browse properties, book site visits, save favorites, and explore listings through an interactive map interface.

Built using **React.js, Vite, Node.js, Express.js, Prisma, MongoDB, and Cloudinary**.

# 📌 Overview

This platform helps users discover and book premium real estate properties.

### 👤 Users Can:

- Browse all properties
- Search properties by location
- View property details
- Book site visits
- Save favorite properties
- Explore properties on map
- Upload profile images

### 🏢 Admin / Property Owners Can:

- Add new properties
- Upload property images
- Manage listings
- View bookings

# 📦 Installation

## Clone Repository

```bash
git clone https://github.com/your-username/real-estate-booking.git
cd real-estate-booking
```

## Install Dependencies

```bash
npm install
```

---

# 🔑 Environment Variables

Create `.env` file:

```env
DATABASE_URL=your_database_url
CLOUDINARY_URL=your_cloudinary_url
JWT_SECRET=your_secret_key
```

---

# ▶️ Run Project

## Frontend

```bash
npm run dev
```

## Backend

```bash
npm start
```
